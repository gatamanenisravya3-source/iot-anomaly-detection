# Anomaly Detection Methods for IoT Applications

Aug 2022 – May 2023 · SJB Institute of Technology

IoT system for detecting weather conditions suitable for plant growth, forecasting weather, and finding change points in collected sensor data.

## What it monitors
Temperature, humidity, pressure, and rainfall — uploaded to the Blynk IoT application for live tracking.

## Tech Stack
IoT, Python, Blynk
